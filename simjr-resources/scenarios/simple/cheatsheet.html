About Sim Jr Cheat Sheets

This is a Sim Jr cheat sheet defined in simple/cheatsheet.html. It's just normal HTML. Links can include embedded Sim Jr JavaScript by using the simjr: scheme like this:

   <a href="simjr:simulation.setPaused(false)">Run</a>
Additional JavaScript functions can be defined in a <script> tag in <head> as usual.

Here are some examples of things you can do:

  • You can run the simulation
  • You can pause the simulation
  • You can center the view on an entity
  • You can select an entity and create a manual controller for it.
  • You can even create links to external sites
